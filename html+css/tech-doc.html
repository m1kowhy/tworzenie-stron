Introduction

JavaScript is a cross-platform, object-oriented scripting language.

It is a small and lightweight language.

console.log('Hello World');

JavaScript contains a standard library of objects.

  • Client-side JavaScript
    • Variables

    You use variables as symbolic names for values in your application.

    let x = 5;

    JavaScript has three kinds of variable declarations.

  • var
  • let
  • const
    • Data Types

    The latest ECMAScript standard defines seven data types:

    let num = 42; // Number

    Six data types that are primitives:

  • Boolean
  • Null
  • Undefined
  • Number
  • String
  • Symbol
    • Functions

    A function definition consists of the function keyword.

    function square(number) { return number * number; }

    Functions can be defined in several ways.

  • Function declarations
  • Function expressions
    • Objects

    Objects in JavaScript can be thought of as collections of key-value pairs.

    let car = { make: 'Toyota', model: 'Camry' };

    Objects can be created in several ways.

  • Object literal syntax
  • Constructor function
    • class Car { constructor(make, model) { this.make = make; this.model = model; } }
